# Telegram Mention All Bot

Простой бот для Telegram, который упоминает всех участников группы.

## Возможности

- `/all` - упомянуть всех участников группы
- `/help` - показать справку
- `@all` в любом сообщении - автоматическое упоминание всех

## Деплой на Render.com

1. Создайте аккаунт на [GitHub.com](https://github.com)
2. Создайте новый репозиторий
3. Загрузите все файлы из этой папки
4. Зарегистрируйтесь на [Render.com](https://render.com) через GitHub
5. Создайте "Web Service" из вашего репозитория
6. Готово! Бот работает 24/7

Подробная инструкция: [`DEPLOY_GUIDE.md`](DEPLOY_GUIDE.md)

## Требования

- Python 3.7+
- python-telegram-bot 20.7

## Важно

⚠️ Бот должен быть администратором группы для получения списка участников.
